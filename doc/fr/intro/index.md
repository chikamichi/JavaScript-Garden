## Introduction

**JavaScript Garden** regroupe un ensemble de documentations sur les parties
les plus bizarres de JavaScript. Vous y trouverez des conseils pour éviter
certains bugs et des erreurs parmi les plus courantes, ainsi que des astuces
pour améliorer les performances du language et éviter certaines mauvaises
pratiques, soit tout un ensemble de problématiques que les développeurs non-
experts ne manqueront pas de rencontrer au cours de leur exploration de
JavaScript.

JavaScriptGarden ne se donne **pas** pour but de vous apprendre le JavaScript.
Des connaissances préalables sont fortement recommandées pour tirer le meilleur
parti de ces différents documents. Si vous découvrez JavaScript et souhaitez
apprendre les bases, vous pouvez consulter en premier lieu l'excellent
[guide][1] disponible sur le *Mozilla Developer Network*.
